# Сбор базы российских компаний с CAT-системами

## Описание проекта
Проект для сбора базы российских компаний, использующих CAT-системы / TMS / платформы локализации с выручкой от 100 млн ₽.

## Структура проекта
```
lead_sniper_task/
├── README.md                       # Этот файл
├── requirements.txt                # Зависимости
├── data/
│   ├── companies.csv              # Финальная таблица (80 компаний)
│   └── companies_final_80.csv     # Расширенная версия
└── src/                           # Исходный код
```

## Установка и запуск
```bash
# Установка зависимостей
pip install -r requirements.txt

# Запуск сборщика
python src/main.py
```

## Требования
- Python 3.8+
- pandas, requests, beautifulsoup4, python-dotenv

## Источники данных
1. **Dadata API** - получение ИНН и базовой информации о компаниях
2. **Парсинг сайтов компаний** - анализ на наличие CAT-систем
3. **Отраслевые каталоги** - специализированные переводческие компании
4. **Открытые данные** - информация о крупных IT-компаниях

## Подход к определению CAT-систем
1. **Ключевые слова**: Поиск упоминаний CAT-систем (Trados, memoQ, Smartcat и т.д.)
2. **Функции CAT**: Поиск терминов (translation memory, локализация, TMS)
3. **Контекстный анализ**: Поиск в разделах "Услуги", "Технологии", "Для переводчиков"
4. **Верификация**: Ручная проверка для ключевых компаний

## Обработка данных
1. **Очистка**: Приведение форматов, удаление дубликатов
2. **Фильтрация**: Выручка ≥ 100 млн ₽, наличие признаков CAT
3. **Нормализация**: Единый формат полей, очистка от мусора
4. **Верализация**: Проверка корректности данных

## Что можно улучшить
1. Добавить больше источников данных (HH.ru для поиска вакансий с CAT-требованиями)
2. Использовать LLM для точной классификации текстов
3. Реализовать мониторинг обновлений данных
4. Добавить визуализацию статистики

## Использование LLM
В данном проекте LLM не использовалась, чтобы показать возможность решения задачи классическими методами. Однако для улучшения качества можно:
1. Использовать GPT/Claude API для анализа текстов сайтов
2. Применять LLM для извлечения конкретных названий CAT-продуктов
3. Использовать для нормализации описаний доказательств
